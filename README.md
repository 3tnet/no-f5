# 安装
```shell
npm i -g no-f5
```
## 使用:
进入项目目录执行
```shell
no-f5 [options]
```
## Available Options:
```shell
`-p` Port to use (defaults to 8080)
```
*现在你可以访问 http://localhost:8080 进入你的应用*
